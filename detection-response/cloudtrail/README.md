# CloudTrail

This folder documents CloudTrail events that are useful for detection and investigation.

## Focus Areas

- console logins
- IAM changes
- policy modifications
- logging disruption attempts
- resource exposure or configuration drift

## Expected Content

- critical event lists
- example detections
- investigation guidance
- correlation ideas with GuardDuty and Security Hub

## Why It Matters

CloudTrail provides essential control plane visibility in AWS. It is foundational for attribution, incident investigation, and cloud detection engineering.
