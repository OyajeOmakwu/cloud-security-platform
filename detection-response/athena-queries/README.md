# Athena Queries

This folder contains example Athena queries for investigating CloudTrail and other AWS log data.

## Use Cases

- review console login activity
- detect policy and permission changes
- identify suspicious access patterns
- investigate public exposure or configuration changes

## Expected Content

- SQL queries for common investigations
- notes explaining what each query helps identify
- examples that support cloud triage and threat hunting

## Why It Matters

Detection engineering is stronger when teams can investigate quickly. Athena queries provide a simple, repeatable way to analyze AWS logs during triage and incident response.
