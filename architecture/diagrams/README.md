# Diagrams

This folder contains architecture diagrams for the cloud security platform.

## Purpose

The diagrams in this folder are used to visualize:

- secure-by-default cloud architecture
- logging, detection, and automated response flows
- policy enforcement points across infrastructure and platforms
- how security capabilities are embedded into developer workflows

## Planned Diagram

The main diagram for this repository will show:

- Terraform-managed AWS infrastructure
- VPC, IAM, and logging foundations
- CloudTrail, GuardDuty, and Security Hub integrations
- EventBridge and Lambda remediation workflows
- Kubernetes policy enforcement with Kyverno and OPA
- CI/CD security controls including SBOM generation, vulnerability scanning, and image signing
