# Image Signing

This folder contains examples related to container image signing and verification.

## Focus Areas

- image provenance
- trusted artifact verification
- deployment trust controls
- integration with CI/CD and policy enforcement

## Expected Content

- signing commands
- verification examples
- notes describing how image trust is enforced

## Why It Matters

Image signing helps ensure that only trusted artifacts move through the delivery pipeline. This is a practical control for reducing software supply chain risk.
