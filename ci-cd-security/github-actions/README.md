# GitHub Actions Security

This folder contains GitHub Actions workflows used to add security validation into CI/CD pipelines.

## Focus Areas

- automated scanning
- security checks on pull requests and pushes
- early detection of risky changes
- support for supply chain security workflows

## Expected Content

- workflow definitions
- scan stages
- enforcement examples
- integration points with SBOM, vulnerability scanning, and signing

## Why It Matters

Embedding security into CI/CD is a key part of DevSecOps. This section demonstrates how security can be integrated into engineering workflows without relying entirely on manual review.
