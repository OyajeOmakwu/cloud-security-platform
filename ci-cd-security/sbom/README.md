# SBOM

This folder contains examples related to Software Bill of Materials generation.

## Focus Areas

- dependency visibility
- supply chain transparency
- artifact inventory
- integration with CI/CD workflows

## Expected Content

- SBOM generation scripts
- example outputs
- notes on how SBOM data supports security review

## Why It Matters

SBOM generation improves visibility into what is actually being built and deployed. This is an important control for modern software supply chain security.
