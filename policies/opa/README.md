# OPA Policies

This folder contains Open Policy Agent examples for enforcing security and compliance rules.

## Focus Areas

- deny privileged workloads
- restrict unapproved registries or patterns
- demonstrate policy-as-code concepts beyond infrastructure provisioning

## Expected Content

- Rego policies
- sample inputs
- denial logic examples
- comparisons with Kubernetes-native policy enforcement approaches

## Why It Matters

OPA is a core policy-as-code technology used across modern platforms. This section shows how security decisions can be standardized, version-controlled, and enforced consistently.
