# IAM Policies

This folder contains example IAM policies used to support cloud security operations.

## Focus Areas

- security audit access
- investigation support
- least-privilege policy design
- reusable identity patterns

## Expected Content

- JSON policy documents
- examples of scoped permissions
- notes describing intended use cases

## Why It Matters

IAM policy design is one of the most important aspects of cloud security. This section demonstrates how permissions can be structured intentionally rather than broadly granted.
