# Istio Security

This folder contains examples related to service-to-service security using Istio.

## Focus Areas

- mutual TLS
- service identity
- zero-trust communication patterns
- policy-based traffic control

## Expected Content

- peer authentication examples
- authorization policies
- notes describing how service mesh security improves workload protection

## Why It Matters

Service mesh security is an important pattern for securing distributed systems. This section demonstrates how identity and encryption can be applied between workloads by default.
