# Kubernetes Security

This folder contains Kubernetes workload examples that demonstrate secure and insecure deployment patterns.

## Focus Areas

- workload hardening
- privileged container prevention
- resource control requirements
- secure-by-default workload design

## Expected Content

- secure pod examples
- insecure pod examples
- policy enforcement demonstrations
- notes explaining risk and mitigation

## Why It Matters

Kubernetes security is a major part of platform security. This section helps show how workload configuration, runtime controls, and policy enforcement work together.
