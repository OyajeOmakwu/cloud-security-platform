# Vault

This folder contains examples and notes related to secrets management using HashiCorp Vault.

## Focus Areas

- dynamic secrets
- secret injection workflows
- reducing reliance on static credentials
- secure application secret access patterns

## Expected Content

- secret injection examples
- usage notes
- explanations of why dynamic secrets are safer than embedded credentials

## Why It Matters

Secrets management is a foundational security capability. This section demonstrates how secret delivery can be designed to reduce exposure and improve operational security.
